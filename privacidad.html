Política de Privacidad

Última actualización: 30 de marzo de 2026

1. Información que recopilamos

Colacho IA recopila únicamente los mensajes que los usuarios envían a través de los canales de atención conectados (WhatsApp, Instagram, Facebook). Esta información se utiliza exclusivamente para generar respuestas automáticas y registrar el historial de conversación en el CRM de la empresa cliente.

2. Uso de la información

La información recopilada se usa para: responder consultas de clientes de forma automática, registrar el historial de conversaciones, y mejorar la calidad del servicio de atención al cliente de la empresa que utiliza Colacho IA.

3. Almacenamiento de datos

Los datos se almacenan en una base de datos local o en servidores en la nube contratados por la empresa cliente. Colacho IA no comparte ni vende información personal a terceros bajo ninguna circunstancia.

4. Datos de terceros

Colacho IA utiliza la API de Meta (Instagram y Facebook) para recibir y procesar mensajes. El uso de estas plataformas está sujeto a las políticas de privacidad de Meta. También utiliza la API de Anthropic para generar respuestas inteligentes y la API de Groq para transcribir mensajes de audio.

5. Derechos del usuario

Los usuarios pueden solicitar en cualquier momento la eliminación de sus datos enviando un correo a: colachosistemaempresarial@gmail.com

6. Contacto

Para cualquier consulta sobre esta política de privacidad contactarse a: colachosistemaempresarial@gmail.com
